# sales-performance-dashboard
An interactive Power BI dashboard tracking corporate sales performance, operating profit margins, and distribution channel analysis.
<img width="1317" height="738" alt="image" src="https://github.com/user-attachments/assets/4adc49a9-2ed7-4b5b-ae7a-cc3ba4aefe24" />
📊 Key Features & Visualizations
* **Total Sales & KPI Cards:** At-a-glance monitoring of core metrics including Total Sales ($401.41K) and Average Operating Margin.
* **Sales Trend Analysis:** A sleek line chart tracking sales volume trajectories month-over-month.
* **Category Breakdown:** High-density horizontal bar charts showcasing quantity distribution across item categories.
* **Profitability Mapping (Scatter Plot):** Advanced scatter chart evaluating the correlation between Operating Profit and Avg Operating Margin by individual product lines.
* **Channel Performance:** Pie and donut visual break-downs contrasting revenue splits across retail Sale Types (Online, Outlet, Direct Sales, Wholesaler).
* **Dynamic Slicers:** Fully interactive dropdown filters for on-the-fly cross-filtering by Category and Date ranges.

## 🛠️ Data Stack & Tools Used
* **Power BI Desktop:** Dashboard architecture, UI theme configuration, and DAX metric formulation.
* **Data Sources:** Combined commercial transaction registers and master product datasets (`InputData` & `MasterData`).
